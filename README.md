# Cactusio
Mi primer gran proyecto en Discord.js, desarrollado en 2021 a los 11 años. Un bot multipropósito con economía, música y diversión. Lo libero bajo GPLv3 como legado educativo y portafolio técnico. Es la base lógica de lo que hoy es Floppa Utilities. ¡Ideal para ver cómo estructurar sistemas complejos desde cero! By Tomcrack.
